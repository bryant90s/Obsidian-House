Obsidian House

Obsidian House is a high-fashion digital atelier concept built around cinematic minimalism, asymmetrical layout systems, and bold editorial contrast.

Inspired by A24 visual language and Maison Margiela’s spatial restraint, the site prioritizes atmosphere, tension, and visual hierarchy over traditional e-commerce clutter.

Design Language

Deep charcoal and true black foundations

Off-white typographic contrast

Metallic silver accent tones

Oversized Playfair Display serif headlines

Dramatic negative space

Layout System

Full-screen overlapping hero typography

Asymmetrical collection blocks

Horizontal scrolling lookbook gallery

Editorial manifesto section

Minimal limited-release grid

Clean, restrained footer

Purpose

This project explores high-fashion digital presence through cinematic pacing, typographic dominance, and controlled spatial imbalance.

It is intentionally bold, restrained, and architectural.

Tech Stack

Vite

TypeScript

Tailwind CSS
